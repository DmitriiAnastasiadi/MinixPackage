# Рабочая тетрадь 4. Задание 2

Создайте пакет для MINIX 3, который будет включать в себя многопоточную программу. Программа должна создавать несколько потоков, каждый из которых выполняет определённую задачу (например, вывод сообщения в консоль). Программа должна быть написана с использованием POSIX-совместимых библиотек, адаптированных для MINIX 3.
